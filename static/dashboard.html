Every consumer message enters Forge, passes through Sentinel's quality gate, gets evaluated by Assay's 4-layer stack, is assigned to an Arena experiment, draws features from Vault, and is monitored in production by Vigil — with a full audit trail from prompt version to conversion outcome.

EVALUATION PIPELINE — ASSAY

Four layers. Zero eyeballing.

Layer 1 — Automated Metrics
ROUGE-1, ROUGE-L, and BERTScore F1 run synchronously on every message. Sub-100ms. First signal.
ROUGE-L BERTScore < 100ms
Layer 2 — Sentinel Classifier
XGBoost model on TF-IDF + sentence embeddings + readability scores. Runs in-memory at < 10ms p99. Hard gate.
XGBoost TF-IDF Embeddings < 10ms
Layer 3 — LLM-as-Judge
Secondary LLM call returns structured JSON: per-dimension Likert scores + reasoning. Catches what classifiers miss.
Structured JSON Async Likert 1–5
Layer 4 — Golden Dataset Regression
Curated (input, expected_range) pairs. Every PR must pass before merge. No silent quality degradation.
